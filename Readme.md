# SlideScribe

Small macOS app to build and run SlideScribe CLI commands through a native SwiftUI interface.

## Features
- Workdir selection via picker or drag & drop
- YouTube URL input
- Advanced options sheet
- Command preview
- Live execution log
- Stop running process

## Notes
This app is intended for personal/local use.
It executes the `slidescribe` command locally, so the binary/script must be available on the machine.
